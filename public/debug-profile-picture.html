Step 5: SQL Query to Run

Copy this and run in Supabase SQL Editor:

-- Check your profile picture URL
SELECT 
  email, 
  first_name, 
  profile_picture_url,
  LENGTH(profile_picture_url) as url_length
FROM profiles 
WHERE email = 'YOUR_EMAIL_HERE';

Step 6: Make Bucket Public

Copy this and run in Supabase SQL Editor:

-- Make avatars bucket public
UPDATE storage.buckets 
SET public = true 
WHERE id = 'avatars';
